# NorthStream Workday Solutions

Enterprise-grade Next.js 14 consulting site architecture for a Workday Integration agency.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Hook Form + Zod
- Framer Motion

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Add shadcn/ui components (if not already added):
```bash
npx shadcn@latest add accordion breadcrumb button card form input navigation-menu select sheet tabs textarea tooltip
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
